# Array Operations & Prime Numbers Practice in C++

This repository contains a **C++ practice project** for learning arrays, loops, functions, and enums.  
The program demonstrates several array operations, including:

- Filling arrays with random numbers.
- Printing arrays.
- Calculating sum, average, max, and min of arrays.
- Counting the frequency of numbers.
- Copying arrays and filtering prime numbers.
- Adding two arrays to create a third array.

This project is mainly for **learning and experimentation purposes**.

## Overview of the File

- **Arrays and Operations:** The program uses three arrays (`Arr1`, `Arr2`, `Arr3`) and performs multiple operations on them.  
- **Random Numbers:** Arrays `Arr1` and `Arr2` are filled with random numbers from 0 to 100.  
- **Prime Numbers:** Only prime numbers can be copied from one array to another.  
- **Array Calculations:** The third array (`Arr3`) contains the sum of `Arr1` and `Arr2` element-wise.  
- **Functions & Enums:** Modular functions are used to make code readable, including `CheckPrimeNumber` using enum `enPrimeNotPrime`.  

## How to Run

1. Download `ArrayAndPrimePractice.cpp`.  
2. Open it in a C++ IDE (Visual Studio, Code::Blocks, VS Code, etc.).  
3. Compile and run the program.  
4. Enter the number of elements for the arrays.  
5. The program will generate random numbers, print arrays, filter primes, and calculate sums.
